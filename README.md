# Projeto-S05
Implementação de um sistema de reserva de salas de estudos para o aplicativo do Inatel.

A funcionalidade "Reserva de Salas" foi desenvolvida para solucionar o problema do aluno ter que ir até a biblioteca toda vez para saber se há ou não uma sala disponível para estudar.
O sistema funciona da seguinte maneira: o usuário entra com a data, o horário de entrada e saída, em seguida o aplicativo processa a disponibilidade de todas as salas, exibindo em vermelho as que estarão ocupadas e em verde as disponíveis. O usuário escolhe a sala desejada e em seguida aparece um resumo da requisição e um botão de confirmação da reserva.
